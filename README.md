# prismagram

Instagram clone with Express + Prisma + React and React Native

## User Stories

-   [x] Create account
-   [x] request Secret
-   [x] Log in
-   [x] Like / Unlike a photo
-   [x] Comment on a photo
-   [ ] Search by user
-   [ ] Search By location
-   [ ] See user profile
-   [ ] Follow / Unfollow User
-   [ ] See the full photo
-   [ ] Edit my profile
-   [ ] Upload photos
-   [ ] Edit the photo (Delete)
-   [ ] See the feed
